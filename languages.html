Python

> Python is a great language to start learning as its syntax is very similar to English.

> Used in data analysis, artificial intelligence, and backend web development, Python is used in many professional companies.

> With the availability of a large range of libraries, Python allows for a wide range of projects.

HTML/CSS

> HTML and CSS can be thought of as an inseperable pair. One is always dependent on the other when it comes to web development.

> HTML can be thought of as the structure and content of a website, while CSS provides all of the stylistic elements.

> Although these languages may seem simple to learn, it still takes a lot of skill to create professional and flawless websites.

Java

> Java is an high-level, object-oriented programming language used to create many web applications.

> The language is very flexible, relatively easy to learn, and used in many large-scale applications.

> However, the language is slowly becoming outdated and used slightly less in today's industry.

C++

> As in the name, C++ is a hybrid language that combines the capabilities of C (a procedural programming language) and object-oriented programming.

> C++ gives you the ability to control the use of your computer's resources and maximize performance.

> However, this language can be difficult to learn at first as the syntax can seem unintuative and requires a lot of precision.

JavaScript

> JavaScript is used for web development and other user-interactive applications.

> Although HTML/CSS may act as the foundation, JavaScript makes a website truly functional and interactive.

> Nowadays, it has been used for mobile/desktop apps as well as game development.
